## Emotion Matrix
Gives mycroft arbitrary emotions

## Description
Give mycroft emotions! using a simple vad matrix, mycroft can remember what emotion he has and change them at your command.

## Examples
 - "Feel happy"
 - "Feel angry"
 - "How are you feeling"
 - "How are you doing"
 - "How you feeling"
 - "How you doing"
 - "Increase arousal"
 - "Decrease valence"
